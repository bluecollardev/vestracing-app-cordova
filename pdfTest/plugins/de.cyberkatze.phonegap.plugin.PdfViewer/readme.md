# Phonegap local PDF-Viewer Android Plugin
    
## Installation
<p><b>cordova plugin add https://github.com/cyberkatze/pdfViewer.git</b></p>
    
## Using:
<p><b>window.openPDF(<fileName_on_SDCARD>);</b></p>
    
## For example:</h3>
<p><b>window.openPDF("/mnt/sdcard" + myfile);</b></p>
